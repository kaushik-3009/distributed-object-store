SYSTEM MATH // GALOIS FIELD MULTIPLICATION

Files are not stored whole. They are split into fragments using Galois Field matrix operations.

k = The exact number of chunks needed to rebuild the file.
n = The total number of chunks distributed across the cluster.
Example: If you upload with k=2, n=5, your file is split into 5 pieces. You can suffer 3 simultaneous server crashes (5 - 2) and still flawlessly recover your data.

RECALCULATION ENGINE: During encode, the file byte vectors are multiplied by a systematic Vandermonde matrix. If fragments are lost, the coordinator extracts the surviving rows of the matrix, performs Gaussian elimination to invert it over GF(2^8), and multiplies the inverted matrix against the surviving chunks to perfectly restore the missing bytes (e.g. D₀ = P₀ ⊕ D₁).
